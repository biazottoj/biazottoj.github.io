Software Maintenance Survey

We are researchers from the Software Engineering Group (SEARCH), from University of Groningen, the Netherlands.

Help us better understand your views on software maintenance tooling. Please consider answering one or more of five short surveys (approx. 5-8 minutes each).

Each survey addresses a specific scenario illustrating a tool usage:

Bot: Label issues based on issue text Dashboard: Present code smells and technical debt IntelliJ IDEA Plugin: Suggest refactoring actions Travis CI Job: Notification based on quality metrics Bot: Facilitate code review in GitHub pull requests

We aim to build and help others build support tools that are driven by the added value for the developer.

Thank you for considering our invitation, and we look forward to your input!
